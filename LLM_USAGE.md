# LLM Usage Declaration

Στο παρόν αρχείο δηλώνεται η χρήση Large Language Models (LLMs) κατά την εκπόνηση της εργασίας.

Η χρήση LLM έγινε υποστηρικτικά για αποσφαλμάτωση, βελτίωση κώδικα, κατανόηση σφαλμάτων και επιλογών του Spark. Οι εκτελέσεις, οι μετρήσεις χρόνων, η συλλογή των αποτελεσμάτων και η τελική αξιολόγηση πραγματοποιήθηκαν από εμάς στο Kubernetes περιβάλλον του μαθήματος.

| Ενότητα / Αρχείο | Εργαλείο που χρησιμοποιήθηκε | Έκταση χρήσης LLM | Τι έγινε από εμάς χωρίς χρήση LLM |
|---|---|---|---|
| Query 3 - DataFrame API | ChatGPT | Βελτίωση του αρχικού κώδικα για τον καθαρισμό της στήλης `Estimated Median Income`, επειδή ορισμένες τιμές εισοδήματος δεν ήταν απευθείας σε αριθμητική μορφή. Συγκεκριμένα, προτάθηκε ασφαλέστερος χειρισμός με έλεγχο μορφής και μετατροπή σε αριθμητικό τύπο. | Η αρχική υλοποίηση με DataFrame API, η επιλογή των στηλών, η σύνδεση των datasets με βάση το `zip_code`, οι εκτελέσεις και η συλλογή των αποτελεσμάτων έγιναν από εμάς. |
| Query 3 - RDD API | ChatGPT | Βοήθεια στον χειρισμό του GeoJSON αρχείου μέσω συνάρτησης `parse_census_line`, ώστε να διαβάζονται σωστά μόνο οι γραμμές που αντιστοιχούν σε Feature objects. | Η γραφή και εκτέλεση του RDD query, η επιβεβαίωση των αποτελεσμάτων και η μέτρηση χρόνου έγιναν από εμάς. |
| Query 3 - Βελτίωση RDD join | ChatGPT | Πρόταση βελτίωσης της RDD υλοποίησης με χρήση `collectAsMap()` και `sc.broadcast()`, επειδή το αρχικό RDD join με shuffle που πραγματοποιήσαμε ήταν ιδιαίτερα αργό. | Η απόφαση να κρατηθεί η βελτιωμένη έκδοση, η εκτέλεση στο cluster και η σύγκριση με την αρχική μορφή έγιναν από εμάς. |
| Join hints - Query 3 και Query 4 | ChatGPT | Δημιουργία βοηθητικής συνάρτησης `apply_join_hint(df, hint_name)`, ώστε το hint να περνάει ως όρισμα από τη γραμμή εντολών και να μην χρειάζεται να αλλάζει χειροκίνητα ο κώδικας του αρχείου πριν από κάθε εκτέλεση. | Οι εκτελέσεις για κάθε hint έγιναν από εμάς στο Kubernetes. Εμείς συλλέξαμε τα physical plans μέσω `explain(mode="formatted")`, τους χρόνους εκτέλεσης και τις join strategies. |
| Γενική αποσφαλμάτωση | ChatGPT | Βοήθεια στην κατανόηση σφαλμάτων των HDFS και Kubernetes συστημάτων. |
| Window functions | ChatGPT | Βοήθεια στην κατανόηση της χρήσης window function στο Spark, ώστε να επιλεγεί η κοντινότερη εγγραφή ανά ομάδα πιο αποδοτικά και κατανοητά. | Η εφαρμογή της τεχνικής στα queries, η εκτέλεση στο Kubernetes και ο έλεγχος των αποτελεσμάτων έγιναν από εμάς. |
